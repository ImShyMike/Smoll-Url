# Smoll-Url
 Simple url shortener made with flask

## Installation
1. Clone the repository:
```bash
 git clone https://github.com/ImShyMike/Smoll-Url.git
```

2. Install dependencies:
```bash
 pip install -r requirements.txt
 ```

## Usage
To run, use:
```bash
 python main.py
 ```
